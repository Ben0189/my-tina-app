# Your First Post

This is a starter template for a new blog post.

## How to use this template

- Change the `title`
- Change the `date`
- Update the `summary`
- Replace this content with your own writing

## Suggested structure

Start with a short introduction that explains what the post is about.

Add a few sections with clear headings.

- Explain the problem
- Show the approach
- Share the result
